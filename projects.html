Water Evaporation Cooler Control System

This project implements a fully functional water evaporation cooler control system using the Arduino Mega 2560 microcontroller. The system integrates multiple hardware components and communication protocols to monitor environmental conditions and control cooling operations automatically.

Communication Protocols & Registers Used:

  • UART - Serial communication (timestamped reporting)
  • ADC - Reading analog values (water level, potentiometer)
  • GPIO - Buttons, LEDs, motor control
  • RTC integration - Timekeeping using the DS3231
